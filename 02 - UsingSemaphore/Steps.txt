1. Run the VM, RockySSH.

2. Access the Semaphore Web Interface
	- Open a browser with your RockySSH IP address on port 3000

		https://yourRockyVM:3000
		

3. Create a Project
	- Specify a Repository (Link to github)
	- Create a Key (SSH or Login)
	- Specify Inventory (Hosts)
	- Assign Tasks (yml)

4. Run Tasks
